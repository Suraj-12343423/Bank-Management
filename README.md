# ATM Machine System in C

A simple ATM Machine project built using C programming language.  
This project allows users to perform basic banking operations through a menu-driven interface.

## Features

- User Login with PIN
- Check Account Balance
- Deposit Money
- Withdraw Money
- Change PIN
- Menu Driven Program

## Technologies Used

- C Programming
- Standard Input/Output Library

## Concepts Used

- Functions
- Loops
- Conditional Statements
- Switch Case
- Global Variables

## How to Run

1. Compile the program

```bash
gcc main.c -o atm
```

2. Run the executable

```bash
./atm
```

## Default PIN

```text
1234
```

## Sample Menu

```text
===== ATM MENU =====
1. Check Balance
2. Deposit Money
3. Withdraw Money
```
